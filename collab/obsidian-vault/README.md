# Project Collaboration Vault

This folder is a project-scoped Obsidian vault tracked in git.

Use it for:

- work items
- design handoff notes
- paper/figma link tracking
- acceptance criteria
- implementation notes

Recommended workflow:

1. Open this folder once in Obsidian Desktop so it becomes a known vault.
2. Enable `Settings > General > Advanced > Command line interface`.
3. Run `./scripts/setup-codex-mcp.sh`.
4. Keep work items in `Work Items/` and use the template in `Templates/`.

Suggested frontmatter keys:

- `id`
- `title`
- `status`
- `paper_url`
- `figma_url`
- `repo_path`
- `branch`
- `owner`
- `updated_at`
