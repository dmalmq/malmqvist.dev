# Rose Pine Figma Tokens

Use this as the exact content model for the `00 Tokens` page in Figma.

## Collections

Create these variable collections:

1. `color`
2. `spacing`
3. `radius`
4. `type`

## Modes

For the `color` collection, create two modes:

- `rose-pine-dawn`
- `rose-pine`

## Page Layout

The `00 Tokens` page should contain four frames:

1. `Color Modes`
2. `Spacing`
3. `Radius`
4. `Type Scale`

Inside `Color Modes`, show both `rose-pine-dawn` and `rose-pine` side by side with swatches and variable names. Do not mix exploratory styles into this page. It should stay reference-only.
