エルスタ病院 (Ersta Sjukhus) — ストックホルム中心部

大規模な医療施設のIFCモデルを、生産対応可能なRevit建築モデルとしてゼロから再構築し、3年間にわたりファサードの調整と複数分野の干渉チェックを主導しました。

導入効果

Monocle Design Awards 2024で「Best Healthcare」を受賞。構造・設備（HVAC）との干渉のないタイムラインを成功裏に調整し、スムーズなファブリケーション（製造）検証へと導きました。

【課題】The Problem

Ersta Nya Sjukhus（新エルスタ病院）は、ストックホルムの急斜面に建設された約23,000㎡の医療施設です。前担当の設計事務所からIFCデータとしてプロジェクトを引き継ぎましたが、IFCはパラメータを持たない「死んだデータ」でした。建設に必要な、完全に統合された「生きたBIMモデル」が必要でした。

【ソリューション】The Solution

提供されたIFCモデルをベースに、Revitモデルをゼロから完全に再構築しました。設計意図をリバースエンジニアリングし、生産可能なシステムへと昇華させ、特徴的なブラウンのアルミファサードシステムのメインモデラーを担当しました。

プロセスとコーディネーション

建物をモデリングすることと、それが実際に建設可能であることを保証することは全く別物です。ファサード担当のコアBIMモデラーとしての3年間、私の焦点は単なるデジタル表現にとどまりませんでした。

Revitモデル（データの源流）

これらの画像はBIMの生データです。特筆すべき複雑さは、建物のボリュームそのものだけでなく、詳細な窓の構成や、38,000㎥の岩盤掘削による地形、そして歴史的建造物との統合にありました。

デザインレビューとオフィスでのワークフロー

Tengbomのオフィスでは、画面を見るだけではなく、デジタルモデルと物理的な素材の検討（特に特徴的な黄土色のアルミクラッディングについて）を常に照らし合わせていました。

建設とファブリケーション

ファサードの製造・施工調整会議に参加することで、設計されたソリューションがRevitモデルの指示通りに製造、設置できることを物理的に検証することができました。

結果

このプロジェクトを通して私が得た最大の学びは、真のBIMマネジメントとは設計意図、生データ構造、そして建設現場の物理的現実を結びつける「結合組織」であるということです。
